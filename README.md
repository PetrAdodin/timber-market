# Timber Market



Timber Market — учебный веб-проект на Django 4.2 для расчетно-графического задания.  

Проект представляет собой современный информационно-каталожный сайт о пиломатериалах.



## Описание



Сайт позволяет просматривать карточки пиломатериалов, фильтровать их по категориям и тегам, открывать детальные страницы материалов, регистрироваться, входить в систему, создавать собственные карточки, оставлять комментарии и ставить лайки или дизлайки.



Тематика проекта: доска, брус, вагонка, фанера, террасная доска, клееный брус, рейка, имитация бруса и другие пиломатериалы.



## Возможности



\- главная страница с каталогом материалов;

\- страница “О компании”;

\- категории материалов;

\- теги материалов;

\- детальная страница карточки пиломатериала;

\- создание пользовательского контента;

\- автоматическая привязка автора к карточке;

\- редактирование карточек только автором или суперпользователем;

\- удаление карточек только автором или суперпользователем;

\- комментарии к карточкам;

\- лайки и дизлайки;

\- регистрация пользователей;

\- авторизация и выход;

\- админ-панель;

\- пагинация;

\- загрузка изображений;

\- собственный современный CSS без Bootstrap.
